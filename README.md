# `bq25895`

[![Crates.io Version](https://img.shields.io/crates/v/bq25895)](https://crates.io/crates/bq25895)
[![docs.rs](https://img.shields.io/docsrs/bq25895)](https://docs.rs/bq25895/latest/bq25895/)

Embedded Rust driver for the BQ25895 battery charge controller.

Written primarily to support [`tildagon`](https://github.com/DanNixon/tildagon-rs/) and as an excuse to give [`device_driver`](https://docs.rs/device-driver/latest/device_driver/) a try.
